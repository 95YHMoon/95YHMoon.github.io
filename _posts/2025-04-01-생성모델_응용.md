# 생성 모델 응용

## Adversarial AutoEncoder (AAE)

- VAE의 KL Divergence 부분을 **GAN**으로 대체
- $q(z \mid x)$를 $p(z)$에 가깝게 만드는 방식의 차이
  - VAE: KL Divergence로 두 분포 간 거리 최소화
  - AAE: Discriminator를 붙여 생성 샘플의 분포를 Target distribution에 맞춤

### VAE와의 비교
- VAE의 KLD: $q$와 $p$ 분포가 **표준 정규분포**라고 가정하고 학습
- AAE: GAN의 특성상 Prior Distribution이 정규분포일 필요 없음
  - Discriminator를 붙여 임의의 Target distribution에 맞출 수 있음

### 구조
- GAN의 Generator 자리 $G(z)$ → Encoder $Q_\theta$로 대체
- KLD 자리가 GAN의 Discriminator로 대체됨

## VAE-GAN

### 개념
- VAE + GAN을 결합한 구조
- VAE의 Decoder 부분이 GAN의 Generator가 됨

### 구조 흐름
```
1. VAE: 진짜 x의 분포로부터 x' 생성
2. GAN: 생성된 x'와 진짜 x를 Discriminator에 입력
         Discriminator를 속이도록 학습
```

### 특성
- GAN이 VAE보다 품질 좋은 이미지를 생성하지만 학습 시간이 더 오래 걸림

## AE / VAE 역할 정리

| 모델 | 역할 |
|---|---|
| AE | 인코더를 효과적으로 학습시키기 위해 Decoder를 붙인 것 |
| VAE | 생성에 목적을 두어, Encoder를 붙였을 뿐 |
