# Word2Vec

> 분포 가정(Distributional Hypothesis)에 기반한 단어 임베딩 기법

---

## 핵심 개념

- 주변 단어(context)의 분포를 통해 단어의 의미를 학습
- 학습 방식: 주변 단어를 보고 중심 단어를 예측하거나, 중심 단어로 주변 단어를 예측

---

## CBOW (Continuous Bag of Words)

> 주변 단어들로 **중심 단어**를 예측

### 학습 방식
- 입력: 왼쪽 단어 + 오른쪽 단어 (각각 one-hot vector)
- ex)
  - (you, goodbye) → say
  - (say, and) → goodbye

### 구조
- 입력층이 두 개 (좌우 각 한 개)
- x_left, x_right를 각각 weighted sum 후 hidden layer 값을 **평균(/2)** 함
- 그 평균 벡터가 임베딩 벡터가 됨

```
x(left)  →  W_in  →  h1
                         → h_avg = (h1 + h2) / 2  →  W_out  →  y_pred
x(right) →  W_in  →  h2
```

---

## Skip-gram

> 중심 단어로 **주변 단어들**을 예측

### 학습 방식
- 입력: 중심 단어 t
- 출력: 주변 단어들 c1, c2, c3, c4

### 예시
```
빨래 | 에서
빨래 | 속옷
빨래 | 를
빨래 | 하는
```

### 핵심
- W_out이 앞뒤 두 개
- 두 개를 평균하여 backpropagation
- 해당 값이 중심 단어(say)의 임베딩 벡터

---

## CBOW vs Skip-gram 비교

| | CBOW | Skip-gram |
|---|---|---|
| 입력 | 주변 단어 | 중심 단어 |
| 출력 | 중심 단어 | 주변 단어 |
| 특징 | 빠른 학습 | 저빈도 단어에 강함 |
